Frequently Asked Questions

Are you affiliated with Valve
No, this is an unoffical app

Is this secure
All communications with the server are encrypted, and no passwords are stored. Steam guard authorizations are stored but can be deleted

Is there a mobile app
Currently there are no mobile apps, but there are plans to develop native iOS and Android apps

I've found a bug/want a feature
Create an issue on GitHub with the bug or feature request

Can I view the code
Yes the code can be found on GitHub, as well as instructions on running your own server
